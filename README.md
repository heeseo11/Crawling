# Crawling
 - 파이썬 Selenium과 BeautifulSoup 활용
 
 | 	|정적 수집	|동적 수집
 |------|---|---|
|사용 패키지|	requests / urllib|	selenium
|파싱 패키지|	beautifulsoup|	beautifulsoup / selenium
|수집 커버리지|	정적 웹 페이지|	정적/동적 웹 페이지
|수집 속도|	빠름 (별도 페이지 조작 필요 X)|	상대적으로 느림
